# 📝 개요
![image](https://github.com/user-attachments/assets/e310dc0a-bffb-48e7-a5a1-c89b08f4610e)

서비스 명: StyleCast

프로젝트 설명: 
 스타일캐스트는 날씨와 패션을 연결하여 사용자의 일상 속 스타일 고민을 해결합니다.
기상 정보와 사용자 데이터를 통합 분석하여 최적의 스타일을 제안하며, 감각적인 UI/UX와 함께 개인 맞춤형 콘텐츠를 제공합니다.
